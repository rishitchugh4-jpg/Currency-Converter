# Currency Converter

A simple Currency Converter web application built using HTML, CSS, and JavaScript. The app fetches real-time exchange rates from a currency API and allows users to convert between different currencies with dynamic country flag updates.

## Features

* Convert between multiple currencies
* Real-time exchange rates using a public API
* Dynamic country flags based on selected currencies
* User-friendly interface
* Input validation
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Currency API
* Flags API
